<h1 align="center">VTS</h1>

<p align="center">竖直标签切换器 - Chrome 快速标签切换，支持缩略图预览</p>

---

## 功能特点

- 按 `Alt+Q` 打开切换器，松开 `Alt` 即可切换
- 每个标签页显示缩略图预览
- 键盘导航：`↑` `↓` `Enter` `Delete` `Esc`
- 悬停显示侧边预览面板
- 自动适配深色/浅色模式
- 支持全屏视频播放

## 安装方法

1. 下载或克隆本仓库
2. 在 Chrome 中打开 `chrome://extensions/`
3. 启用右上角的**开发者模式**
4. 点击**加载已解压的扩展程序** → 选择 `VTS` 文件夹

## 使用方法

| 按键 | 操作 |
| ---- | ---- |
| `Alt+Q` | 打开切换器 / 向下导航 |
| `Alt+Shift+Q` | 向上导航 |
| `↑` / `↓` | 浏览标签 |
| `Enter` | 切换到选中标签 |
| `Delete` / `Backspace` | 关闭选中标签 |
| `Esc` | 关闭切换器 |

松开 `Alt` 键即可切换到选中的标签页。

## 设置说明

点击扩展图标切换设置：

| 选项 | 描述 |
| ---- | ---- |
| 预览 | 在列表中显示缩略图 |
| 特写 | 显示侧边预览面板 |

## 常见问题

**问：为什么有些标签没有缩略图？**  
答：Chrome 限制了对内部页面（chrome://、edge:// 等）的截图。

**问：可以修改快捷键吗？**  
答：可以，访问 `chrome://extensions/shortcuts`。

**问：隐身模式下能用吗？**  
答：可以，在扩展详情页启用"在隐身模式下启用"。
