<h1 align="center">VTS</h1>

<p align="center">Chrome 快速标签切换器，支持预览</p>

---

## 功能亮点

- 按 `Ctrl+Space` 打开标签切换器
- 每个标签页显示缩略图预览
- 键盘导航：`↑` `↓` `Enter` `Delete` `Esc`
- 点击切换标签，悬停显示侧边预览
- 自动适配深色/浅色模式
- 支持标题覆盖模式

## 安装方法

1. 下载此文件夹
2. 在 Chrome 中打开 `chrome://extensions/`
3. 启用右上角的**开发者模式**
4. 点击**加载已解压的扩展程序** → 选择 `vertical_tab_switcher` 文件夹

## 使用方法

| 按键 | 操作 |
| ---- | ---- |
| `Ctrl+Space` | 打开/关闭切换器 |
| `↑` / `↓` | 浏览标签 |
| `Enter` | 切换到选中的标签 |
| `Delete` / `Backspace` | 关闭选中的标签 |
| `Esc` | 关闭切换器 |

## 设置说明

点击扩展图标访问设置：

| 选项 | 描述 |
| ---- | ---- |
| 预览 | 在列表中显示缩略图 |
| 覆盖 | 在缩略图上显示标题覆盖 |
| 特写 | 显示侧边预览面板 |

## 常见问题

**问：为什么有些标签没有缩略图？**  
答：Chrome 限制了对某些页面（chrome://、edge:// 等）的截图。

**问：可以修改快捷键吗？**  
答：可以，访问 `chrome://extensions/shortcuts` 进行自定义。

**问：隐身模式下能用吗？**  
答：可以，在扩展详情页启用"在隐身模式下启用"。
